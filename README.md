# Modern Storage — Self Storage Hub

Consumer-facing self-storage hub for Modern Storage® at https://self-storage.modernstorage.com/.

## Stack

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- Bebas Neue + system font stack (matches podcast.modernstorage.com)

## Site architecture

- `/` — Hub homepage (this project)
- `/climate-controlled` — Climate-controlled storage
- `/boat-rv-vehicle` — Boat, RV, and vehicle storage
- `/business` — Business and mini-warehouse storage
- `/household-moving` — Household storage and moving

## Local development

```bash
npm install
npm run dev
```
